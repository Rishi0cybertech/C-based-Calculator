🧮 C-based-Calculator

A simple menu-driven calculator program written in C language that performs basic arithmetic operations.
This project is created as a beginner practice project to understand user input, swtich-case statements, and arithmetic operations in C programming.

📌 Features
The calculator currently supports the following operations :
➕ Addition
✖️ Multiplication
➗ Division
➖ Subtraction
The program asks the user to input two numbers and choose an operation from the menu.

⚙️ Technologies Used
C language concepts :
* veriable
* User Input ('scanf')
* Output ('printf')
* Arithmetic Operations
* 'swtich case' Statements

 🧠 How It Works
 1. The user enters two integers.
 2. A menu of operations is displayed.
 3. The user selects the desired operation.
 4. The program calculates the result using a 'switch case' control structure.
 5. The result is displayed on the screen.

📋 Menu Example
for addition choose 1
for multiplication choose 2
for division choose 3
for subtraction choose 4

⚠️ Current Limitations

* Division by zero is not handled.
* The calculator performs only one calculation per run.
* Only integer operations are supported.
